# ROTN v1.0 - Rot Notation Specification

## [IT] Specifica Italiana

**ROTN** è un formato testuale ASCII per la descrizione di posizioni scacchistiche accessibili.
Progettato per la lettura con screen reader NVDA. Alternativa alla FEN.

**Principi:**

1.  Leggibile da umano e macchina.
2.  Definito formalmente in BNF EBNF/ISO 14977.
3.  Estensione file: `.rotn`

**Descrittori Obbligatori - Ordine fisso:**
`T;V;B;N;M;P;`

| **Tag** | **Significato**     | **Valori**                             |
| ------- | ------------------- | -------------------------------------- |
| **T**   | Titolo/Tema         | Stringa ASCII                          |
| **V**   | Tratto              | `B` = Bianco, `N` = Nero               |
| **B**   | Posizione Re Bianco | `Rh1..Ra8`                             |
| **N**   | Pezzi Neri          | Lista `Pezzohx` separati da `,`        |
| **M**   | Pezzi Bianchi       | Lista `Pezzohx` separati da `,`        |
| **P**   | Punteggio/Tema      | `1`=Matto, `0`=Patta/Stallo, `n`=Mosse |

**Notazione Pezzi:** `R`=Re, `D`=Donna, `T`=Torre, `A`=Alfiere, `C`=Cavallo, `P`=Pedone.

### [IT] Esempi

**Esempio 1: Matto in una mossa**

````rotn
T:Matto in una mossa;V:N;B:Rh8;N:Rf7;Dg1;M:N;P:1;
Lettura NVDA: `Titolo: Matto in una mossa. Tratto Nero. Re Bianco H8. Nero: Torre F7. Bianchi: Donna G1. Matto in 1.`

*Esempio 2: Stallo*
T:Stallo;V:B;B:Ra1;N:Rc2,Dd3;M:;P:0;

## [EN] International Specification

*ROTN* is an ASCII text format for accessible chess positions.
Designed for NVDA screen reader. FEN alternative.

*Principles:*
1. Human and machine readable.
2. Formally defined in EBNF/ISO 14977 BNF.
3. File extension: `.rotn`

*Mandatory Descriptors - Fixed Order:*
`T;V;B;N;M;P;`
**Tag** | **Meaning** | **Values**
**T** | Title/Theme | ASCII String
**V** | Side to move | `B` = White, `N` = Black
**B** | White King Position | `Kh1..Ka8`
**N** | Black Pieces | List `Piecegx` comma separated
**M** | White Pieces | List `Piecegx` comma separated
**P** | Score/Theme | `1`=Mate, `0`=Stalemate/Draw, `n`=Moves
*Piece Notation:* `K`=King, `Q`=Queen, `R`=Rook, `B`=Bishop, `N`=Knight, `P`=Pawn.

### [EN] Examples

*Example 1: Mate in 1*
T:Mate in 1;V:N;B:Kh8;N:Rf7;Qg1;M:N;P:1;
*Example 2: Stalemate*
T:Stalemate;V:B;B:Ka1;N:Kc2,Qd3;M:;P:0;
````
