# ROTN v1.0 - Rot Notation Specification

**Documento Ufficiale:** [2026-07-02-Spec-ROTN.md](https://redyouman.github.io/blog/2026/07/02/Spec-ROTN.html)

## [IT] Specifica Italiana

**ROTN** è un formato testuale ASCII per la descrizione di posizioni scacchistiche accessibili.
Progettato per la lettura con screen reader NVDA. Alternativa alla FEN.

**Principi:**

1.  Leggibile da umano e macchina.
2.  Definito formalmente in BNF EBNF/ISO 14977.
3.  Estensione file: `.rotn`

**Descrittori Obbligatori - Ordine fisso:**
`T;V;B;N;M;P;`
Obbligo: ogni descrittore e i suoi valori devono stare su una riga propria.

| **Tag** | **Significato**     | **Valori**                                  |
| ------- | ------------------- | ------------------------------------------- |
| **T**   | Titolo/Tema         | Stringa ASCII                               |
| **V**   | Punto di vista      | `B` = Bianco, `N` = Nero                    |
| **B**   | Posizione Re Bianco | `Rh1..Ra8`                                  |
| **N**   | Pezzi Neri          | Lista `Pezzohx` separati da `;`             |
| **M**   | Pezzi Bianchi       | Lista `Pezzohx` separati da `;`             |
| **P**   | Mossa               | Numero di mossa a cui è arrivata la partita |

**Notazione Pezzi:** `R`=Re, `D`=Donna, `T`=Torre, `A`=Alfiere, `C`=Cavallo, `P`=Pedone.

### [IT] Esempi .rotn

**Matto in una mossa**

````rotn
T:Matto in una mossa;
V:N;
B:Rh8;
N:Rf7;Dg1;
M:N;
P:1;
_Stallo_
T:Stallo;
V:B;
B:Ra1;
N:Rc2;Dd3;
M:B;
P:0;
```
````

## [EN] International Specification

_ROTN_ is an ASCII text format for accessible chess positions.
Designed for NVDA screen reader. FEN alternative.

_Mandatory Descriptors - Fixed Order:_
`T;V;W;B;M;P;`
Mandatory: each descriptor and its values must be on its own line.
| **Tag** | **Meaning** | **Values** |
| ------- | ----------------------- | ----------------------------------- |
| **T** | Title/Theme | ASCII String |
| **V** | Side / Color| `W` = White, `B` = Black |
| **W** | White King Position | `Kh1..Ka8` |
| **B** | Black Pieces | List `Piecegx` semicolon separated |
| **M** | White Pieces | List `Piecegx` semicolon separated |
| **P** | Move | Move number the game has reached |
_Piece Notation:_ `K`=King, `Q`=Queen, `R`=Rook, `B`=Bishop, `N`=Knight, `P`=Pawn.

### [EN] Examples .rotn

_Mate in 1_
T:Mate in 1;
V:B;
W:Kh8;
B:Rf7;Qg1;
M:B;
P:1;
_Stalemate_
T:Stalemate;
V:W;
W:Ka1;
B:Kc2;Qd3;
M:W;
P:0;
