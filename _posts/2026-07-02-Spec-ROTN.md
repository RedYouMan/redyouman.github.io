# ROTN v1.1 - Rot Notation Specification

## [IT] Specifica Italiana

**ROTN** = **Ro**t **N**otation  
Formato testuale ASCII per descrivere posizioni scacchistiche accessibili.

**Regole:**

1.  **Estensione**: `.rotn`
2.  **Ordine Fisso**: `T;V;B;N;M;P;`
3.  **1 riga = 1 tag**

**Descrittori Obbligatori - Ordine fisso:**
`T;V;B;N;M;P;`

| **Tag** | **Significato**             | **Valori / Esempio**                          |
| ------- | --------------------------- | --------------------------------------------- |
| **T**   | Testo della posizione       | `T:Matto in 2 mosse;`                         |
| **V**   | Punto di vista / Lato       | `V:B;` = Guardi dal Bianco, `V:N;` = dal Nero |
| **B**   | Posizione Pezzi Bianchi     | Lista `Pezzoxx` separati da `;`               |
| **N**   | Posizione Pezzi Neri        | Lista `Pezzoxx` separati da `;`               |
| **M**   | Chi muove                   | `M:B;` oppure `M:N;`                          |
| **P**   | Numero di mossa di partenza | `P:1;`                                        |

**Notazione Pezzi:** `R`=Re, `D`=Donna, `T`=Torre, `A`=Alfiere, `C`=Cavallo, `P`=Pedone.  
**Coordinate:** notazione algebrica `a1..h8`

### [IT] Esempio Posizione Iniziale

```rotn

V:B;
B:Ta1;Cb1;Af1;Dd1;Re1;Af2;Cb2;Th1;Pa2;Pb2;Pc2;Pd2;Pe4;Pf2;Pg2;Ph2;

M:B;

## [EN] International Specification v1.1

_ROTN_ is an ASCII text format for accessible chess positions.

_Mandatory Descriptors - Fixed Order:_
`T;V;B;N;M;P;`
**Tag** | **Meaning** | **Values / Example**
**T** | Position Text | `T:Mate in 2;`
**V** | Viewpoint / Side | `V:W;` = View from White, `V:B;` = from Black
**B** | White Pieces Position | List `Piecexx` ; separated
**N** | Black Pieces Position | List `Piecexx` ; separated
**M** | Move Side | `M:W;` or `M:B;`
**P** | Starting Move Number | `P:1;`
_Piece Notation:_ `K`=King, `Q`=Queen, `R`=Rook, `B`=Bishop, `N`=Knight, `P`=Pawn.

### [EN] Example Starting Position
T:Starting Position;
V:W;
B:Ra1;Nb1;Bf1;Qd1;Ke1;Bc2;Ng2;Rh1;Pa2;Pb2;Pc2;Pd2;Pe4;Pf2;Pg2;Ph2;
N:Ra8;Nc8;Bb5;Qd8;Ke8;Bf5;Nc6;Rh8;Pa7;Pb7;Pc7;Pd7;Pe5;Pf7;Pg7;Ph7;
M:W;
P:1;

Finito. Vai.
```
